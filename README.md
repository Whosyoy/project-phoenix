# Project Phoenix

> 从 Java 平台工程师，到企业级 AI Agent 工程师

**像工程师一样构建 AI，而不只是使用 Prompt。**

## 项目愿景

Project Phoenix 是一个围绕真实 DevOps / CI-CD 场景展开的企业级 AI Agent 学习、实践与沉淀项目。

它记录我从 Java 后端 / 研发平台工程师，向 AI Native / AI 应用开发 / 企业级 Agent 工程师转型的完整过程。

## 为什么做这个项目

AI 应用开发不只是写 Prompt，也不只是做 Demo。

真正可落地的企业级 AI 应用，需要同时具备：

- 真实业务理解
- 扎实的软件工程能力
- Tool Calling / MCP 工具化能力
- RAG 与知识工程能力
- Agent 规划、编排与安全边界设计
- Evaluation / 评测体系
- 权限、审计、幂等、限流、灰度、可观测性等生产级工程能力

本项目希望基于真实研发效能 / CI-CD 场景，系统化沉淀这些能力。
