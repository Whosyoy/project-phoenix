# Project Phoenix

## 1. Vision

From Java Platform Engineer to Enterprise AI Agent Engineer.

Build AI like an Engineer, not just a Prompt User.

## 2. Goal

在 4 个月内，基于真实 DevOps / CI-CD 场景，完成一个企业级 AI Agent 的系统化学习、设计、实践和面试资产沉淀。

最终目标：具备投递 AI Native / AI 应用开发 / AI 平台研发岗位的项目能力和面试表达能力。

## 3. Core Principles

### Principle 1: Business First

所有学习都必须映射到 DirectBus 真实业务场景。

### Principle 2: Engineering First

目标不是做 Demo，而是按企业级工程方式设计 AI 应用。

### Principle 3: Understand Before Coding

代码可以由 AI 辅助生成，但架构、边界、取舍必须自己理解。

### Principle 4: Architecture Before Framework

先明确业务问题、架构和数据流，再选择 Spring AI、Ragent、LangGraph 或其他框架。

### Principle 5: Output Every Week

每周必须沉淀至少一个可展示成果：设计文档、架构图、Demo、评测集、面试回答或复盘。

## 4. Success Criteria

- 能完整讲清 DirectBus Agent 的业务背景、架构设计和技术取舍。
- 能解释 Agent、MCP、RAG、Evaluation、Production Engineering 的完整生命周期。
- 有可展示的 Demo 或源码改造记录。
- 有完整的项目材料、架构图、面试问答和简历描述。
- 能开始投递 AI Native / AI 应用开发 / AI 平台研发相关岗位。
